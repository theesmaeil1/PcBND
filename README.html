🚀 PcBND Bot - ربات تلگرام برای دانشجویان کامپیوتر بندرعباس

PcBND Bot یک ربات تلگرام قدرتمند و کاربرمحور است که برای دانشجویان رشته کامپیوتر بندرعباس طراحی شده است. این ربات امکان به‌اشتراک‌گذاری جزوات، پروژه‌ها، و درخواست‌های آموزشی را فراهم می‌کند و با ابزارهای مدیریتی پیشرفته، به ادمین‌ها اجازه می‌دهد تا منابع و کاربران را به‌راحتی مدیریت کنند. این پروژه با استفاده از Python، کتابخانه pyTelegramBotAPI، و پایگاه داده SQLite ساخته شده است.

🌟 چرا PcBND Bot؟

  • جامعه‌محور: فضایی برای همکاری و اشتراک دانش بین دانشجویان.
  • مدیریت ساده: ابزارهای قدرتمند برای نظارت و مدیریت منابع.
  • امنیت بالا: احراز هویت با شماره تلفن و تأیید منابع.
  • رابط کاربری جذاب: منوهای تعاملی و دکمه‌های شیشه‌ای.

🛠 نصب و راه‌اندازی

📋 پیش‌نیازها

  • Python 3.8 یا بالاتر
  • کتابخانه pyTelegramBotAPI
  • پایگاه داده SQLite (به‌صورت پیش‌فرض در Python موجود است)

🔧 مراحل نصب

  1. کلون کردن مخزن: 
    git clone https://github.com/<your-username>/PcBND-Bot.git
cd PcBND-Bot
  2. نصب وابستگی‌ها: 
    pip install pyTelegramBotAPI
  3. تنظیم متغیرها:

    فایل bot.py را باز کنید و مقادیر زیر را تنظیم کنید:

    • BOT_TOKEN: توکن ربات از BotFather
    • GROUP_CHAT_ID: آیدی گروه تلگرام
    • ADMIN_ID: آیدی عددی ادمین
  4. اجرای ربات: 
    python bot.py

⚠️ نکات مهم

  • ربات باید در گروه تلگرام ادمین باشد و دسترسی‌های لازم را داشته باشد.
  • توکن ربات را در متغیرهای محیطی ذخیره کنید.
  • اتصال پایدار به اینترنت ضروری است.

📖 راهنمای استفاده

  1. شروع کار:

    دستور /start را ارسال کنید و شماره تلفن خود را با دکمه 📱 اشتراک شماره تلفن تأیید کنید.

  2. منوی اصلی:
    • 📤 آپلود فایل: آپلود PDF با عنوان درس و استاد.
    • 📚 منابع آموزشی: مشاهده و دانلود منابع.
    • 📩 درخواست‌ها: ارسال درخواست به ادمین.
    • ℹ️ راهنما: اطلاعات ربات و لینک‌ها.
    • 🛠 پنل ادمین: ابزارهای مدیریتی (فقط ادمین).
  3. پنل ادمین:

    مدیریت منابع، ارسال پیام، و مشاهده اطلاعات کاربران.

📂 ساختار پروژه
